# /onboard - 신규 팀원 온보딩

프로젝트의 전체 맥락을 한눈에 파악할 수 있도록 종합 정보를 제공합니다.

## 워크플로우

1. `mcp__archflow__archflow_onboarding_context` 호출
2. 결과를 아래 구조로 정리하여 답변

## 응답 구조

### 1. 프로젝트 개요
- 프로젝트가 무엇인지 한 줄 요약
- 사용 기술 스택 (GitHub 레포 언어 기반)

### 2. 시스템 아키텍처
- 주요 컴포넌트와 연결 관계
- Draw.io 다이어그램 기반 설명

### 3. 현재 진행 상황
- 현재 스프린트의 핵심 작업
- 진행률 요약

### 4. 시작하기
- 어떤 레포를 클론하면 되는지
- 핵심 파일/디렉토리 안내
