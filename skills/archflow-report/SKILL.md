# /report - 팀 활동 보고서

비개발자(대표님, PM)도 이해할 수 있는 주간 활동 보고서를 생성합니다.

## 워크플로우

1. `mcp__archflow__archflow_team_activity` 호출
2. `mcp__archflow__archflow_jira_sprint_status` 호출 (보충)
3. 결과를 보고서 형태로 정리

## 보고서 구조

### 이번 주 완료된 작업
- 완료된 이슈 목록 (비기술 언어로)
- 머지된 PR 수

### 진행 중인 작업
- 현재 작업 중인 항목
- 담당자별 정리

### 주요 지표
- 완료율 (%)
- 이슈 처리 건수
- PR 머지 건수

### 다음 주 예정
- 남은 스프린트 이슈
- 예상 일정

## 톤앤매너
- 기술 용어 최소화
- 비즈니스 임팩트 중심으로 설명
- 숫자와 퍼센트 활용
